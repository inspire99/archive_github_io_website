Top 10 Programming Languages 2019

Photo by Arif Riyanto on Unsplash

Do you want to check whether your programming skills are still up to date or to make a strategic decision about what programming language should be adopted when starting to build a new software system.

Is there a metric which will allow to determine popularity of programming languages?

One day or the other, we are exploring. We are exploring different possibilities. Sometimes what we are used to , might help. But, there might be better ways of getting things done. That's when we might have to re-evaluate our toolkit to solve problems.

  • Enter TIOBE index. 

TIOBE relies on search engines. What results are displayed by them. 

From TIOBE,

The TIOBE Programming Community index is an indicator of the popularity of programming languages. The index is updated once a month. The ratings are based on the number of skilled engineers world-wide, courses and third party vendors. Popular search engines such as Google, Bing, Yahoo!, Wikipedia, Amazon, YouTube and Baidu are used to calculate the ratings. It is important to note that the TIOBE index is not about the best programming language or the language in which most lines of code have been written.


Source: https://www.tiobe.com/tiobe-index/

  1. Java 
    1.  powers lot of backend technologies which we see today
  2. C
    1. powers kernels of OSes, embedded devices
  3. Python 
    1. powers lot of machine learning runtime ( imagine numpy processing the vectors related to images )
  4. C++ 
    1. powers device drivers, game engines
  5. C#
    1. most of applications using microsoft products are powered by C#
  6. Visual Basic .NET
    1. similar to C#, powers windows console apps, windows based services
  7. JavaScript
    1. after advent of node-js ( or v8 ), its everywhere now.
  8. PHP
    1. Facebook & other websites
  9. Objective-C
    1. powers applications on Apple devices
  10. SQL
    1. backend databases are queried using this language. 

TIOBE index is not the only one method to determine popularity of programming languages.

  • Enter Redmonk
Redmonk's judgement depends on Github & StackOverflow

From Redmonk,

As always, these are a continuation of the work originally performed by Drew Conway and John Myles White late in 2010. While the specific means of collection has changed, the basic process remains the same: we extract language rankings from GitHub and Stack Overflow, and combine them for a ranking that attempts to reflect both code (GitHub) and discussion (Stack Overflow) traction. The idea is not to offer a statistically valid representation of current usage, but rather to correlate language discussion and usage in an effort to extract insights into potential future adoption trends.

What is the next big thing for the software world?


Photo by Samuel Zeller on Unsplash

What's in for future in software world?
Cloud ensures system availability always. ( Cloud computing, distributed systems)
Accelerators ensures processing by small sensors. ( IOT )
Sensory experiences by interaction with mixed reality. ( AR, VR, XR , Haptic, Smart display)
Robots, drones, machine interaction with humans ( AI, machine learning )
Connectivity for the devices ( Networking , 5G.. 6G )
What's next in for the technology world?
I guess we ( computer scientists) might need to start looking at research aspects from other fields like physics, chemistry, biology. As they are going to determine the future, as they did in past and present.
  • Cloud Computing
Has Cloud Computing reached its saturation point?  Anything which happens over the internet could be considered as cloud computing, as long as it satisfies the criteria / factors enabling availability, scalability, fault-tolerance. Imagine we can hook up our mobile devices and provide / share the mobile computing ? Do we have such enablement? Cloud Computing space seems to be ever-evolving in its own pace & dynamic need.
  • Machine Learning
What about Machine Learning? We do have algorithms based on mathematics, solving the fundamental problem of machine learning to think like a human. Machines can see, speak, analyze, run ( boston dynamics ). Machine Learning is combining with Cloud Computing space to provide API endpoints to analyze data. It also has gone further in federated learning to download a model onto a device, do some learning, upload the local model to the cloud, to improve accuracy of global model. This too is ever-evolving, thanks to recent advancements in hardware & programming. 
  • Mixed Reality
Augmented Reality seems to have caught craze of few demographic groups. It provides an interface, over our 3D space, to interact with virtual objects. Pokemon allowed humans to learn about Augmented Reality. Lot of virtual worlds could be created, combined with virtual reality, to provide a totally different interactive experience. This one is also advancing in its pace, but there's little consumer adoption.
  • Drones
Drones for delivery? How will common people consume drones technology? Drones have helped analyze spatial situations, where humans could not possibly navigate physically. Drones combined with Augmented reality could provide an experience for humans to reach out to places, seek data/ provide feedback.
  • Networking
Networking is synonymous with connectivity, routing. Whatever be the medium ( wired/ wireless), connectivity is essential for all the above to materialize. Application Layer depends on Networking Layer.

Electricity?

Lastly, technology fundamentally depends on electricity ( charged particles ). How is the innovation space with electricity, Imagine a virtual world and electricity generated within virtual world is being used to power-up the real world? Is this theoretically possible?
